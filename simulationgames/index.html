The Simulation Games

Setup

To set up a game, DM the command sg!setup to the bot.
Note that this will fail if someone else is setting up a game, or if a game is currently in progress.
The bot will then guide you through setup; take notice of the comment at the bottom of the tribute list.

Starting the Game

To start the game, send the command sg!start to the channel you want the game to take place in.
Only the person who set up the game can do this.

Progressing the Game

To progress the game, send the command sg!next to the channel the game is taking place in.
This can only be done by the person who set up the game, and can only be done in the channel the game was started in.

Stopping Mid-Game

To stop a game which is in progress, send the command sg!stop to the channel it is taking place in.
